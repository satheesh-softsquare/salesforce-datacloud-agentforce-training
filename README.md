# Salesforce Data Cloud & Agentforce — Hands-On Training

A structured, hands-on training program covering Salesforce Data Cloud (Data 360) and Agentforce.
Each use case is self-contained, domain-specific, and has three progressive difficulty levels.

## Difficulty Levels (within each use case)

- Beginner — Data ingestion, DMO mapping, Identity Resolution
- Intermediate — Calculated Insights, Segmentation, Activation, Data Actions
- Advanced — Search Index, Retriever, Prompt Builder, Agentforce Agent

## Prerequisites

- Salesforce Developer Org or SDO with Data Cloud enabled
- Data Cloud Admin or Operator permission set
- Agentforce and Einstein Studio enabled (for Advanced level)

## How to Use

Each use case folder contains two documents:
1. Facilitator Guide — Full solutions, hints, common mistakes. For the trainer only.
2. Participant Handout — Challenge card (pages 1-2) + detailed tasks. Share page 1-2 first.
